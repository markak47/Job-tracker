# Job-tracker
A full-stack job application tracker built with React, TypeScript, Node.js, and MongoDB.

reason for project choice:

while searching for project to build my experience i decided to create a project that will benefit my personally and will also be able to challenge my skills. i setteled for job tracker application to know my overall status of the job applications, and manage easily how i can improve myself by looking at my application procedure.



## Features
- User Authentication (JWT)
- Track job application status
- Interview reminders
- Filter, search, and stats dashboard

## Tech Stack
- Frontend: React + Tailwind + TypeScript
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- Deployment: Vercel (frontend), Render (backend), MongoDB Atlas
